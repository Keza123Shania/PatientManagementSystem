# Patient Management System

A Java console application designed to manage clinic patient data using Object-Oriented Programming (OOP) principles and efficient data management strategies.

## Key Features

### OOP Fundamentals
- Implements **Inheritance**, **Abstraction**, and **Polymorphism**
- Models core entities such as **Doctors** and **Patients** in a scalable and maintainable way

### Data Management
- Replaces static arrays with the **Java Collections Framework** for flexibility and performance:
    - **Lists**: Maintain an ordered history of appointments
    - **Sets**: Ensure a unique registry of doctors
    - **Maps**: Enable fast patient lookups using unique IDs

###  Error Handling
- Implements **custom exception hierarchies**
- Includes validation logic to:
    - Prevent system crashes
    - Ensure data integrity
    - Provide meaningful error feedback
